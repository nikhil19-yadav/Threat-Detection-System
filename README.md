# 🛡️ Threat Detection System

An AI-based surveillance system that detects potential threats using computer vision and monitors surrounding devices using Bluetooth and WiFi scanning.

---

## 🚀 Features
- 🔍 Real-time object detection using YOLOv8
- 📶 WiFi network scanning
- 🔵 Bluetooth/BLE device detection
- 📷 Live camera monitoring
- ⚡ Interactive UI using Streamlit

---

## 🛠️ Tech Stack
- Python
- OpenCV
- YOLOv8 (Ultralytics)
- Streamlit
- PyWiFi
- Bleak

---

## 📂 Project Structure
- app.py  
- bluetooth_scanner.py  
- wifi_scanner.py  
- drone_detector.py  
- yolov8n.pt
